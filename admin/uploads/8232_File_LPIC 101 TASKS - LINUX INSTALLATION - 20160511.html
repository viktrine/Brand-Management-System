LPIC 101 TASKS - LINUX INSTALLATION - 20160511

TASK 1 - GNU/LINUX INSTALLATION Install the following GNU/LINUX Operating System a) CENTOS 6.8 LIVE LINUX b) CENTOS 6.8 ENTERPRISE LINUX c) REDHAT 6.8 ENTERPRISE LINUX TASK 2 - GRUB CONFIGURATION a) Configure the GRUB to load the above GNU/LINUX operating systems b) Change the boot manager in the MBR from CentOS 6.8 to RedHat 6.8 and vice-versa TASK 3 - CONFIGURATION OF LAN AND INTERNET SETTINGS a) Configure the Local Area Networks (LAN) settings. Assign your machine a static IP Address b) Configure the Internet settings. Make sure your machine can access the Internet. c) Enable all the trusted services in the firewall as well as the interfaces. TASK 4 - ADDING REPOS a) Add the following repositories to your system to facilitate the installation of software packages using the YUM tool. b) Install gparted software package using the YUM tool. TASK 5 - LOADING OPERATING SYSTEMS DIRECTLY USING GRUB MENU Attempt to load the installed operating systems directly using the GRUB menu.
